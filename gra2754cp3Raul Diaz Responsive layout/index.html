Our Story

This is a responsive restaurant page established for my school project. The goal is to show a responsive design using modern HTML and CSS Skills and frameworks. I used simple Italion Flag Colors Red, White, and Green and a simple layout to make an Italian Pizza restaurant site with Responsive Elements.

The food is based on classic Italian recipes, focusing on thr ingredients and preparation. The layout is designed to be fully Responsive so it can work well on mobile, tablet, and desktop screens.
